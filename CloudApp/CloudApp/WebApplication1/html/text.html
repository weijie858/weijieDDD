  • -把代码存储到git仓库中
  • 1 把代码放到仓库的门口
  • git add ./文件
  • git add ./ 全部文件放入仓库门口
  • 2把仓储门口的代码放到里面的房间中去
  • git commit -m "备注信息" 提交到仓库房间
  • 3 查看状态 可以查看当前代码是否放入仓库
  • git status
  • 4 一次性把代码放到仓库的房间里（版本库）
  • git commit --all -m "备注信息" 全部修改文件提交的仓库房间
  • 5 git查看日志
  • git log --oneline
  • git中忽略文件
  • gitignore 在这个文件夹中可以设置要被忽略的文件或者目录。
  • 被忽略的文件不会被提交到仓库里
  • 在.gitignore中可以书写要被忽略的文件的路径，以/开头，
  • 一行写一个路径。这些路径所对应的文件都会被忽略，
  • 不会被提交到仓库
  • 写法：
  • * '/.idea' 会忽略.idea文件
  • * '/js' 会忽略js目录里的所有文件
  • * '/js/*.js' 会忽略js目录里的所有js文件
  • git reset --hard Head~0
  • git reset --hard [版本号versionnum]
  • git reflog
  • -12流行框架-git创建分支，切换分支
  • 默认是有一个主分支master
  • 创建分支 git branch [分支名]
  • 查看分支 git branch
  • 切换分支 git checkout [分支名]
  • 在主分支中执行 合并分支 git merge [分支名]
  • 在主分支中执行 git branch -d [分支名] 删除指定分支
